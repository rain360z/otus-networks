## 3 Включим ISIS, сконфигурируем оборудование.

Приведём дефолтные настройки

```
feature isis

router isis Underlay
net 47.[xxxx.xxxx.xxxx].00
address-family ipv4 unicast
log-adjacency-changes

is-type level-1
```
